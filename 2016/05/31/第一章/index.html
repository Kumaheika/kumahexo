使用 Hexo 來寫 Blog 吧

node.js Hexo sass?

本篇主要紀錄安裝的過程以及遇到的問題

之前有聽聞同事 Jacky 在前端社群中，介紹用 gitpage 來架設個人的 blog，一直都很有興趣想要試試看，剛好原本的 WP 網域空間也到期，藉這個機會來嘗試一下，順手紀錄架設的過程與遇到的問題～

step 01 - 完成 Hexo 需求環境

安裝 Hexo 之前，請先安裝好 Node.js 以及 Git

step 02 - 安裝 Hexo 2.8.3

這邊是聽同事建議，2.8.3 似乎比較安定，所以選擇這個版本
在安裝好 Node.js 後，接著執行 command line，輸入：

如果遇到權限問題，前面請加上 sudo

step 03 - 建立要安裝 Hexo 的資料夾

這邊可以自己找個地方新增資料夾，也可以用 cmd

step 04 - 前往 Hexo 資料夾並安裝

在 cmd 中，先移動到剛剛新增的資料夾內

接著執行 Hexo 的初始化，並安裝node

然後安裝 Hexo 的 plugin

step 05 - 確認是否安裝、並執行

到你安裝 Hexo 的資料夾內，打開 package.json 檔案，
裏面會有你安裝的 plugin 的名稱，有的話表示成功。
接著，要在本機上啟動 Hexo

然後開啟你的瀏覽器，輸入 localhost:4000 就可以看到你的預設首頁啦!!
